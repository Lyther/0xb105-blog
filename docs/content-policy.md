# Content Policy

Publish work that is reproducible, useful, and ethically scoped.

Required for research and CVE posts:

- Scope and authorization model
- Affected versions or tested commit hashes
- Reproduction environment
- Root cause
- Impact
- Patch, mitigation, or detection guidance
- Disclosure timeline when applicable

Avoid publishing instructions that enable unauthorized access, persistence, credential theft, or destructive activity against real third-party systems.
